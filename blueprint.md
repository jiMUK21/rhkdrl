# Project Blueprint

## Overview

This project is a simple lottery number generator. It generates 6 unique random numbers between 1 and 45.

##- Current Plan

1.  **Modify `index.html`**
    *   Add a title for the lottery number generator.
    *   Create a container to display the generated numbers.
    *   Add a button to trigger the number generation.
    *   Link to the `style.css` and `main.js` files.
2.  **Create `style.css`**
    *   Style the main container, the number display area, and the button.
    *   Use a modern and visually appealing design.
    *   Ensure the layout is responsive.
3.  **Modify `main.js`**
    *   Implement the logic to generate 6 unique random numbers.
    *   Add an event listener to the button to generate and display the numbers when clicked.
